reduce <- function(x) {
  repeat{
    if(x %% 2 == 0){
      print(x)  
      x<-x/2  
    }
    else{
      return(x)
      break
    }
  }
}

Default Arguments

As you probably know by now, many functions come with default arguments, which can be left unspecified without affecting the execution of the function.

The following function attempts to compute the length of the hypotenuse of a right triangle, but will accept one or two leg lengths.

pythagoras <- function(a, b = a) {
  return(sqrt(a^2 + b^2))
}

pythagoras(3, 4)
## [1] 5
pythagoras(5)
## [1] 7.071068

Ellipsis ()

Finally, in addition to normal and default arguments, the ... argument allows a function to take any number of arguments.

The following function simply counts the number of arguments given.

countargs <- function(...) {
  print(paste("This function received", length(list(...)), "arguments."))
}

countargs(0)
## [1] "This function received 1 arguments."
countargs(1L, 5, 3 + 5i, TRUE, "yosemite")
## [1] "This function received 5 arguments."
countargs(c(1, 2, 3, 4, 5), factor(c("one", "two", "three")))
## [1] "This function received 2 arguments."

More commonly, however, the ... is simply passed as optional arguments to another function.

Your turn! Write a function which produces a histogram of a random process. The first argument should be n, the number of observations. The second argument should be one of the following: "Uniform", "Binomial", "Poisson". These should correspond, respectively, to: runif(n), rbinom(n, 100, 0.5), rpois(n, 1). Finally, include the ... argument to be passed into the histogram function.

# Remove eval = FALSE or set eval = TRUE when you have completed this function.
randhist <- function(n, dist, ...) { # Fill in arguments.
  if(dist == 'Uniform'){
    hist(runif(n),...)
  }
  else if(dist == 'Binomial'){
    hist(rbinom(n, 100, 0.5), ...)
  }
  else if(dist == 'Poisson'){
    hist(rpois(n,1), ...)
  }
  else{
    stop('one of Uniform, Binomial, or Poisson should be  given')
  }
  # Fill in.
}
